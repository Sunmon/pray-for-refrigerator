# pray-for-refrigerator

냉장고에 있는 음식을 입력하면 요리를 추천해주는 서비스

## What is different?

냉장고를 부탁해 ; 냉장고속 재료 아무거나 입력하면 가능한 음식 추천해줌. 자취생맞춤!

ex ) 후추 검색 => 후추가 들어가는 음식 추천

만개의레시피 ; 이름에 재료가 꼭 들어가야함

ex) 후추 검색 => 후추 ㅇㅇㅇㅇ 레시피 나옴
